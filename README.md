# iAttend — Attendance Intelligence Platform

A multi-organization attendance management SaaS built with React + Firebase.

## Features
- 🔐 Organization sign up & login
- 📁 Upload member lists via Excel or CSV
- ✅ Multi-gate real-time check-in
- 📋 Full attendance records per session
- 📊 Analytics with charts and trends
- ⬇️ Download attendance as Excel
- ⚙️ Customizable gate names

## Tech Stack
- React 18
- Firebase (Auth + Firestore)
- Recharts (analytics)
- SheetJS / PapaParse (file handling)

## Deploy on Netlify
1. Push this repo to GitHub
2. Go to netlify.com → New Site → Import from Git
3. Build command: `npm run build`
4. Publish directory: `build`
5. Deploy!

## First Time Setup
1. Sign up with your organization name and email
2. Go to Members → upload your Excel or CSV file
3. Go to Settings → rename your gates
4. Go to Check In → start your first session!
